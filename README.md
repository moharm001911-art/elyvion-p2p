# Elyvion P2P v2 - MVP (Next.js 14 compatible)

Quick start:
1. unzip the archive
2. npm install
3. npm run dev
4. Open http://localhost:3000

Features:
- Arabic / English (user chooses in Settings)
- Light / Dark theme toggle in Settings
- Pages: landing, add-offer, offer, dashboard, admin, settings
- API route simulating escrow
- Next.js 14 Link usage fixed

Env (optional):
- ADMIN_EMAIL
